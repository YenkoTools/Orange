## About

The Orange Service is built using Clean Architecture principles with .NET 10. This project demonstrates modern software development practices including:

- **Clean Architecture** - Separation of concerns with Domain, Application, Infrastructure, and API layers
- **Modern .NET** - Built on .NET 10 RC2
- **Logging** - Integrated Serilog for structured logging
- **Testing** - Comprehensive xUnit test coverage
- **API Documentation** - OpenAPI/Swagger integration

## Quick Start

```bash
# Clone the repository
git clone https://github.com/jkulba/Orange.git
cd Orange

# Run the application (default port 5001)
./run.sh

# Or specify a custom port
PORT=3000 ./run.sh
```
